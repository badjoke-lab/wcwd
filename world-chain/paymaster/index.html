World Chain Paymaster Preflight

Check World Chain RPC readiness and prepare a sponsor request template.

WCWD uses a fixed RPC for the network check. Sponsor endpoints are never fetched or posted to by WCWD servers.

Tool role

Network check

Reads chain ID and gas price through WCWD's fixed World Chain Mainnet RPC path.

Sponsor template

Builds a copyable curl command locally. WCWD does not send it.

Storage

Public preflight GET requests are read-only and are not added to Worker history.

Local sponsor template

Sponsor / Paymaster Endpoint

Used only to build a local curl template.
