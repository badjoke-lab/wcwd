Oracle Feed Tester

Read an AggregatorV3 feed through WCWD's fixed World Chain RPC path.

The server does not accept caller-selected RPC URLs, does not follow redirects, and does not save public feed checks.

Tool role

Current state

Same-origin feed check using WCWD's fixed World Chain Mainnet RPC.

Server boundary

Only the feed contract address is accepted. External RPC URL input is rejected.

Storage

Public GET requests are read-only. Results are returned to this page and are not added to history.
