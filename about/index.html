About WCWD

What is WCWD?

WCWD (World Chain / World ID Dashboard) is a lightweight, builder-focused toolkit for the World ecosystem.

It provides a text-first dashboard for monitoring World Chain network conditions and token-related metrics, along with practical tools to help integrate and validate World ID.

The interface avoids heavy UI patterns and wallet connections, focusing instead on clarity and speed — so you can check what you need and move on.

All features are currently available for free.

What WCWD Provides

WCWD is structured around two main areas:

  • World Chain dashboards
  • World ID integration tools

World Chain Section

/world-chain/

A high-level dashboard for viewing World Chain network conditions.

  • TPS, gas metrics, and address statistics
  • Quick WLD price reference
  • Sample token ratio indicators
  • Ecosystem project listings

The dashboard is designed to remain lightweight and avoid unnecessary indexing dependencies wherever possible.

/world-chain/sell-impact/

A simple simulation tool to estimate the potential price impact of selling WLD.

  • Input sell amount
  • View estimated slippage
  • Approximate impact calculation

This tool provides rough estimates only and does not constitute financial advice.

/world-chain/oracles/

A reference page for oracle data available within the World Chain ecosystem.

  • Price feed listings
  • Data source overview
  • Implementation considerations

Designed as a practical reference for builders selecting oracle feeds.

/world-chain/paymaster/

A structured overview of gas abstraction and paymaster concepts.

  • Core paymaster principles
  • Example use cases
  • Implementation considerations

This section functions primarily as documentation and architectural guidance.

World ID Section

/world-id/

An overview of World ID integration and access to related tools.

World ID enables proof of personhood. WCWD focuses on helping builders implement and validate it effectively.

/world-id/wizard/

An integration wizard that generates World ID implementation snippets.

  • Environment selection (Web, Next.js, Node/Express, Cloudflare Workers, etc.)
  • Frontend snippet generation
  • Backend verification example code
  • Environment variable templates

All output blocks include copy buttons for immediate use.

/world-id/debugger/

An offline proof JSON analyzer for World ID.

  • Structural validation
  • Missing field detection
  • Basic format checks
  • Categorized diagnostic hints

This tool performs local structural checks only and does not call external APIs.

/world-id/playground/

A verifier endpoint testing tool.

  • Generates curl, fetch, and axios request examples
  • Allows manual testing against your own verifier endpoint
  • Displays response output
  • Clearly explains CORS failures when applicable

Designed for local and development-stage validation.

Design Philosophy

WCWD is built around the following principles:

  • Text-first design
  • Static architecture
  • No wallet dependency
  • Lightweight and fast
  • Builder-oriented
  • Not an official replacement for World documentation

This site is an independent reference project and does not provide financial advice or official support.

Who This Is For

  • Developers building on World Chain
  • Builders integrating World ID
  • Anyone who wants a concise overview of ecosystem status

How WCWD Is Run

WCWD is currently fully free to use.

It is maintained as an independent project without ads or subscription models.

Beyond basic hosting and infrastructure costs, future improvements — such as enhanced historical data or more precise analytics — may require external indexing services or third-party data providers.

These enhancements are intended to make WCWD a more practical and reliable tool for builders.
