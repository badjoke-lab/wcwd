# 📘 **Worldchain × WLD Dashboard — 完全仕様書 v1（無料版 / AI封印版）**

---

# 1. **プロダクト概要**

**Worldchain × WLD Dashboard** は、Worldcoin エコシステムの中でも
「無料でデータが取得できる範囲」に限定して構築する、
**完全無料運用の統計ダッシュボード**。

以下の方針を絶対条件とする：

### **🟥 費用条件（絶対ルール）**

* **運営費 0 円で動作すること**
* AI推論は **一切使用しない（許可が出るまで完全封印）**
* バックエンド（DB / cron / サーバー）は **0円運用の範囲に限定**
* 無料APIのみ使用可能
* Cloudflare Pages または GitHub Pages で完結する静的サイト

### **🟥 拡張条件**

* 拡張はすべて **無料で増やせる範囲から順に実装**
* 費用が発生する機能は「あなたが課金OKと言った瞬間にのみ」開放
* AI活用部分は **仕様書に封印セクションとして明記**

---

# 2. **非機能要件**

### **● 無料運営**

* サーバー費 0円
* API費用 0円（無料APIだけを使用）
* AI費用 0円（AI封印）
* DB費用 0円（DB不使用）

### **● 稼働環境**

* Cloudflare Pages（推奨）
* GitHub Pages（代替）

どちらも完全無料。

### **● パフォーマンス**

* 全データ取得はクライアント側の fetch のみ
* 1回の更新によるAPIリクエストは「数回以内」に抑える

### **● 可観測性**

* localStorage を使用し、
  “前日 → 今日” の差分データを保存・比較（サーバー不要）

### **● セキュリティ**

* APIキー不要のエンドポイントを優先
* 必要な場合は “client-safe”（公開して問題ないキー）に限定する

# 3. **使用可能な API（すべて無料）**

### ■ Worldscan

Worldchain のブロックチェーンエクスプローラ（無料API利用）

取得可能：

* 最新ブロック
* ブロック内 TX 数
* ガス情報（baseFee など）
* アドレス数（推計）

### ■ QuickNode Gas Tracker（閲覧無料）

* Worldchain のガス価格（推定値）

### ■ CoinGecko

* WLD の価格・前日比・24h出来高
* 7d / 30d ヒストリカル価格

### ■ （補助）他の無料エクスプローラ

* Blockscout Worldchain
* 過去データ取得が可能な場合のみ利用

## ⚫ **4.1 Worldchain Network 基本指標**

すべて無料APIから取得。

### 表示項目

* **TPS（現在値）**
* **24h TX 総数**
* **新規アドレス数（24h）※推計**
* **総アドレス数（推計）**
* **ガス価格（現在値）**

### 更新頻度

* ページ読み込み時（初期）
* ユーザーが「更新」ボタンを押したとき

※自動ループ禁止（API制限回避目的）

## ⚫ **4.2 WLD Market 指標**

CoinGecko（無料）を使用。

表示項目：

* **WLD 現在価格（USD/JPY）**
* **前日比（%）**
* **時価総額**
* **24h 出来高**
* **7d スパークライン**

---

## ⚫ **4.3 日次差分（localStorage）**

AI不使用のまま文章整形を行う。

保存するデータ：

* 左記すべての主要指標の前日値

表示例：

* 「TX +14.8%」
* 「新規アドレス +5200」
* 「価格 -2.1%」

サーバー不要・無料。

---

## ⚫ **4.4 Activity Breakdown（軽量分類）**

TXの type / method を取得し、自前で分類：

* Native Transfer
* Token Transfer
* Contract Call
* Mint / Burn
* Bridge系（推測分類）

AI不要・運営費ゼロ。

---

## ⚫ **4.5 異常検知（閾値ベース）**

AIなしで実装できる範囲。

例：

* TPS > 過去平均 × 1.4 → **Spike**
* TX < 過去平均 × 0.7 → **Drop**
* ガス > 通常値 + 50% → **High Gas**

※通知は UI 表示（バナー or カード内の警告）だけ
→ 外部通知は「無料でできないので」封印。
